Testify - Rose Hack 2025

An AI assistant for instructors to handle their question papers for quizes and exams.

GitHub Repository DevPost

About the Project

Developed a Python-based backend leveraging Flask for a web application with integrated OpenAI API for advanced text analysis. Implemented file upload and processing features, including PDF generation and ZIP archive handling, to support scalable and efficient workflows

Key Features

  • Can control the randomeness of the questions (Backend feature only)
  • Created a UI to upload the files and retireve the generated material.
  • Log-in feature to secure previous question papers generated

Technologies Used

  • Python: Core logic and development.
  • Flask: Connection to the front-end application.
  • OpenAI API: AI model to process the lecture material and provide the questions.
  • Git: Version control for seamless team collaboration.

Key Accomplishments

  • First hackathon and successfully implimented the Backend and the Frontend.
  • Enhanced skills in APIs and text analasys.

Future Goals
